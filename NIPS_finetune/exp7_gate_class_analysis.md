## 结论

1. **Frequency gate 偏差确认为系统性**: 7A 平均 -20%, 7B 平均 -13%, 两种正则化下都存在。7B (consistency) 偏差更轻
2. **Temporal gate 无偏差**: 两组 Δ < 0.1%, 可忽略
3. **Top10 ratio 反而 abnormal 略高**: abnormal 的信息更集中在 top 特征上 (+0.4~0.6%), 虽然 freq gate 整体更低但集中度更高
4. **7B seed 间方差更大**: s2025 偏差 -31% 但 s3407 几乎无偏差, 说明 consistency 正则化效果不稳定, sparse 更一致地存在偏差
