# Pretrained Weights

The pretrained model weights of CodeBrain are available on Hugging Face:

https://huggingface.co/YjMajy/CodeBrain/tree/main

Please download the checkpoints from the above link and place them in this directory.

- `CodeBrain_Tokenizer.pth` is the pretrained weights for the `TFDual_Tokenizer`
- `CodeBrain.pth` is the pretrained weights for the `EEGSSM`
